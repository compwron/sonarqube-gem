+ (Proc) sonarqube(server_url, username = '', password = '')

Functional interface for this gem. Creates a curried proc, which can be called using other functional style methods as parameters. The returned curried Proc initialises/contains a RestClient::Resource object with all the necessary connectivity information

Examples:

sq = SonarQube.sonarqube 'localhost:9000', 'admin', 'admin'

def self.sonarqube(server_url, username='', password='')
  
  if [username, password].reduce(:+) == ''
    connector = RestClient::Resource.new server_url
  else
    connector = RestClient::Resource.new server_url, username, password
  end
  
  proc {|endpoint| connector[endpoint].get}.curry
end
