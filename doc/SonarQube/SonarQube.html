Overview

Initializes a connector object (RestClient::Resource), that stores the url and optionally the username and password for the SonarQube server.

Examples:

sq = SonarQube.new 'localhost:9000', 'username', 'password'

def initialize server_url, username='', password=''
  if [username, password].reduce(:+) == ''
    @connector=RestClient::Resource.new(server_url)
  else
    @connector=RestClient::Resource.new(server_url, username, password)
  end
end
